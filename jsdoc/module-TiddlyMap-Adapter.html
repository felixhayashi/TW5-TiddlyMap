Class: Adapter

Class: Adapter

Adapter

new Adapter(wiki)

This library acts as an abstraction layer above the tiddlywiki system. All the provided methods give the api-user the chance to interact with tiddlywiki as if it was a simple graph database.

Everything that is related to retrieving or inserting nodes and edges is handled by the adapter class.

You don't need to create your own instance of this class. The adapter service may be accessed from anywhere using $tw.tmap.apapter.
